# Daklink – Nền tảng truy xuất nguồn gốc nông sản hữu cơ

Daklink là giải pháp minh bạch hóa chuỗi cung ứng nông sản, giúp kết nối người nông dân tận tâm với người tiêu dùng thông thái thông qua công nghệ truy xuất nguồn gốc hiện đại.

## Tính năng nổi bật

- **Truy xuất nguồn gốc**: quét mã QR để xem chi tiết hành trình của sản phẩm từ nông trại đến tay bạn.
- **Câu chuyện nông dân**: Hiểu thêm về quy trình canh tác và tâm huyết của người sản xuất.
- **Minh bạch thông tin**: Mọi bước trong quy trình chế biến đều được ghi nhận rõ ràng.

## Công nghệ sử dụng

Dự án được xây dựng trên nền tảng công nghệ hiện đại, đảm bảo tốc độ và trải nghiệm người dùng mượt mà:

- **Frontend**: React, TypeScript, Vite.
- **Styling**: Tailwind CSS, shadcn/ui.
- **Animations**: Framer Motion.

## Phát triển cục bộ

Để chạy dự án ở môi trường local, bạn cần cài đặt Node.js và thực hiện các bước sau:

1. Cài đặt dependency:
   ```sh
   npm install
   ```

2. Chạy môi trường development:
   ```sh
   npm run dev
   ```

3. Build sản phẩm:
   ```sh
   npm run build
   ```
